# Digital Computer Skill System

A web-based system designed to track student skills, progress, and performance in computer courses, helping teachers manage and evaluate learning effectively.

---

## 🚀 Features
- Student skill tracking  
- Progress monitoring  
- Score/marks management  
- Simple and user-friendly interface
